# Global Research Trends on Heavy Metal Contamination in Bivalve
## Purpose: Reproducible publication data availability

This repository has been created to ensure transparency, reproducibility, and accessibility of the research workflows used in the study.

## Contents

- `Raw/` – Raw bibliometric data as exported from WoS.  
- `code/` – R script used for bibliometric processing and figures.  
- `Data/` – Cleaned/standardized datasets used in final analysis.  
- `README.md` – Overview of the repository, its purpose, and instructions.

## Usage

This repository is intended for academic and research purposes. Users can:

1. Explore the bivalve and metal bibliometrix datasets.  
2. Run the analysis scripts.  
3. Reproduce the figures and results reported in the corresponding manuscript.

note: The repository will be updated with the final citation once the manuscript is accepted.
